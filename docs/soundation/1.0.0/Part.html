Part

Part

Part is a sequence of sounds that play respectively

Constructor

new Part(partData, partName)

To create a part you need to provide an array of degrees to play the corresponding note in the chord that has been set. E.g 0.1 [ { time: '4n 0', degree: 0, duration: '4n' }, { time: '4n 1', degree: 1, duration: '4n' }, { time: '4n 2', degree: 2, duration: '4n' }, { time: '4n 3', degree: 3, duration: '4n' }, ]
