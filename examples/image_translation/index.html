A library to hear what you cannot see

Soundation is a library that helps translating image to audio. This can be used to help visually impaired or colorblind people. It has been developed as a mini project for my Thesis and was primarily build to test such scenario. It lacks a lot of features due to it's origin but it is build with extensibility in mind.

Bpm manipulation based on color

Bpm translation

When you hover the mouse on the picture soundation grabs the color and translates it to a bpm value. You should hover smoothly and wait for the sound change. When the sound is steady, click to learn the color.

Track change based on color

Track translation

When you hover the mouse on the picture soundation grabs the color and translates it to a new track. The moment you hear a track change stop and click to learn the color.
